## 🚀 Run Locally

**Prerequisites:** Node.js (v18+)

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Start the development server:**
   ```bash
   npm run dev
   ```

3. **Open Application**
   Visit `http://localhost:5173` in your browser.

## 🌍 Deployment Ready

This application is built with React, Vite, and TailwindCSS (with Framer Motion). It is fully optimized for static deployment.

**Deploy to Vercel (Recommended):**
1. Push this repository to your GitHub account.
2. Sign in to [Vercel](https://vercel.com).
3. Import the repository. Vercel will automatically detect `Vite` settings.
4. Click **Deploy**. Your customized Axonyx platform will go live in under a minute.

**Connecting a Custom Domain (E.g., Hostinger):**
1. Inside Vercel, go to the project **Settings > Domains**.
2. Input your custom domain (e.g., `axonyx.com`).
3. Follow the DNS instructions (usually adding an A record pointing to `76.76.21.21` or setting the nameservers inside your Hostinger dashboard).
